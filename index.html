¿Qué es una Base de Datos?

Imagina un archivador digital muy organizado. En lugar de carpetas, tiene "tablas" donde se guarda la información de forma estructurada.

2 / 10

¿Qué es una Tabla?

Es como una hoja de cálculo. Tiene filas (registros) y columnas (campos) para organizar los datos. Por ejemplo, una tabla de "Usuarios".

+----+---------+-------------------+
| ID | Nombre  | Email             |
+----+---------+-------------------+
| 1  | Ana     | ana@email.com     |
| 2  | Luis    | luis@email.com    |
+----+---------+-------------------+
3 / 10

¿Qué es SQL?

SQL (Structured Query Language) es el idioma que usamos para comunicarnos con la base de datos. Le damos órdenes como "dame estos datos" o "guarda esta información".

4 / 10

CREATE TABLE: Crear una Tabla

Usamos este comando para construir la estructura de una tabla, definiendo sus columnas y el tipo de dato que cada una guardará.

-- Así se crearía la tabla Usuarios:
CREATE TABLE Usuarios (
    ID INT,
    Nombre VARCHAR(100),
    Email VARCHAR(100)
);
5 / 10

SELECT: Consultar Datos

Es el comando más usado. Sirve para leer o seleccionar datos de una tabla. Se combina con FROM para indicar de qué tabla.

-- Selecciona todos los datos de la tabla Usuarios
SELECT * FROM Usuarios;
6 / 10

WHERE: Filtrar Resultados

La cláusula WHERE te permite poner condiciones para obtener solo los registros que necesitas.

-- Selecciona solo al usuario con ID = 1
SELECT * FROM Usuarios WHERE ID = 1;
7 / 10

INSERT INTO: Añadir Datos

Se utiliza para agregar nuevas filas (registros) a una tabla.

-- Agrega un nuevo usuario a la tabla
INSERT INTO Usuarios (ID, Nombre, Email)
VALUES (3, 'Carlos', 'carlos@email.com');
8 / 10

UPDATE: Modificar Datos

Permite cambiar valores en registros que ya existen. ¡Cuidado! Siempre usa WHERE para no modificar toda la tabla.

-- Cambia el email del usuario con ID = 2
UPDATE Usuarios
SET Email = 'luis.nuevo@email.com'
WHERE ID = 2;
9 / 10

Resumen y Próximos Pasos

¡Felicidades! Ya conoces los 4 pilares de SQL (CRUD):

SELECT (Leer), INSERT (Crear),
UPDATE (Actualizar), DELETE (Borrar).

Con esto, ya puedes realizar las operaciones más importantes en cualquier base de datos. ¡El siguiente paso es practicar!
